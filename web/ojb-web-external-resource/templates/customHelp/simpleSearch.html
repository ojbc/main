SIMPLE SEARCH TUTORIAL

By default, the 'Simple Search' screen will start with the source systems available for the query selected. The user must uncheck any systems that will not be a part of the search.

Once the source system(s) are set, the user must type the search parameter(s) into the box initially labeled 'Search term(s)'. Some of the valid parameter combinations are:

  • Last name only
    If the last name contains spaces or hyphens, combine all into 1 word. For example, if searching for the last name "De Los Santos", enter "DeLosSantos" into the parameter box. As mentioned in '2. Search Parameters - Person Search', last name searches only use 'sounds like' qualifiers currently.
    *May return "System CJIS HI returned too many records, please refine your criteria." The Criminal History search only allows 250 returns maximum. If this error is shown, the user must refine the search by adding a first name parameter.

  • First and last name
    The format for this combination is 'John Doe' (FirstName LastName).

  • Social security number
    The format for this parameter is nnn-nn-nnnn.

  • Driver's License number
    The format for this parameter is Hnnnnnnnn
    *This parameter is currently only valid for the Warrants source system, using this parameter to search the Criminal History source system will cause the "too many records" error.

  • State Identification number
    The format for this parameter is annnnnnn, 1 letter followed by 7 numbers.

  • FBI number
    The format for this parameter is 9 alphanumeric characters.

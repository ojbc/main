SUBSCRIPTIONS



The Subscription Management service enables users to subscribe to events and receive notifications when those events occur.

Creating a Subscription

To create a subscription, from the Subscription screen, select the Add Subscription button.

Editing a Subscription

To edit an existing subscription, from the Subscription Management screen, select the Edit button to the left of an existing subscription.

Deleting a Subscription

To delete an existing subscription, from the Subscription Management screen, select the Unsubscribe button to the right of an existing subscription.

The sub-topics will cover the different types of subscriptions.

Person Search Parameters



First Name

Also known as "Given Name". User has the ability to specify:
  • "Starts With": Results in subjects with the first name searched, as well as subjects with the first name plus additional characters. For example, a "Starts With" search on 'Bob' will return 'Bob', and then 'Bobby', etc.
  • "Exact Match": Results in subjects with only this exact first name. For example, an "Exact Match" search on 'Bob' will return only subjects with 'Bob' as their first name.

Middle Name

This parameter uses a "Starts With" qualifier by default.

LAST NAME

User has the ability to specify:
  • "Starts With": Results in subjects with the last name searched, as well as subjects with the last name plus additional characters. For example, a "Starts With" search on 'Johns' will return 'Johns', and then 'Johnson', etc.
  • "Exact Match": Results in subjects with only this exact last name. For example, an "Exact Match" search on 'Johns' will return only subjects with 'Johns' as their last name.

SID #, FBI #, SSN, DL/ID #

All of these parameters are valid identifiers. Only exact matches will return a result.

SID #

State Identification numbers are usually formatted with 1 letter, followed by 7 digits.

FBI #

Federal Bureau of Investigation numbers are typically 9 alpha numeric characters.

SSN

Social security numbers should be searched using the format xxx-xx-xxxx.

DL/ID #, Issuer

The drive license number for a person.
